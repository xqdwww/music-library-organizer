# Contributing

Small fixes, tests, and documentation improvements are welcome. Please open an issue before proposing a new format or workflow so the project can remain low-maintenance and local-only.

Contributions must not add media downloading, service authentication, scraping, DRM circumvention, remote storage management, real music fixtures, or user data. Run the unit tests and `ruff check .` before submitting a pull request. By contributing, you agree that your contribution is licensed under GPL-2.0-or-later.
